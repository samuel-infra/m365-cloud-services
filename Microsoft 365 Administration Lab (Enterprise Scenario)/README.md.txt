# Microsoft 365 Tenant Administration

## Purpose
This project demonstrates how I set up and managed a Microsoft 365 environment for a small organization.

---

## Scenario
I simulated a company environment where I:
- Created users and assigned roles
- Configured security (MFA)
- Set up collaboration tools (Teams & SharePoint)
- Managed email and basic administration

---

## Licensing & Setup
- Used Microsoft 365 Business Premium (trial)
- Created a tenant (demotconsulting)
- Configured basic environment settings

---

## Users & Groups
- Created multiple users
- Assigned licenses
- Created:
  - Security Groups
  - Microsoft 365 Group (Team-Operations)
- Assigned roles (User Administrator)
- Enabled MFA for security

---

## SharePoint & Intranet
- Created a SharePoint communication site
- Built a simple intranet homepage
- Added:
  - Document library
  - Ticket system (list)

---

## Teams & Exchange
- Created a Team (Team-Operations)
- Connected it to Microsoft 365 Group
- Managed mailbox in Exchange Admin Center

---

## Security & Troubleshooting
- Configured MFA
- Checked permissions and group access
- Explored Microsoft Purview
- Simulated troubleshooting scenarios
